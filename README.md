# Cairo Weather Data Analysis

## Project Overview

This project analyzes historical weather data in Cairo from 2009 to 2024 to investigate temperature trends and evaluate whether there is a significant increase over time.

## Research Question

Has Cairo's average temperature increased sigmificantly between 2009 and 2024?

## Hypothesis

H0: There is no significant trend in temperature over the years in Cairo

H1: There is significant trend in temperature over the years in Cairo

## Dataset

Source: Open-Meteo Historical Weather API via Kaggle

Time period: 2009 - 2024

Number of observations: 5,813

Variables used: Temperature, Date

## Tools used

Python (pandas, matplotlib, seaborn, scipy)

## Analysis Performed

Data cleaning and preprocessing

Aggregation into yearly averages

Visualization:
- Scatter plot
- Line chart
- Regression plot
- Seasonal comparison

Hypothesis testing using linear regression

## Key Results

Temperature shows an overall increasing trend

Later years have higher average temperatures than earlier years

Summer temperatures increased more significantly than winter

Statistical testing showed p-value < 0.05, so H0 is rejected
